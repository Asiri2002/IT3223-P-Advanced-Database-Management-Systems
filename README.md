# IT3223 (P) Advanced Database Management Systems

## Overview

This repository contains practical exercises, assignments, and projects completed for the **IT3223 (P) Advanced Database Management Systems** module.

The module focuses on advanced database concepts, database design techniques, query optimization, transaction management, and modern database technologies.

## Topics Covered

* Advanced SQL Queries
* Database Design and Normalization
* Indexing Techniques
* Views and Stored Procedures
* Triggers and Functions
* Transactions and Concurrency Control
* Query Processing and Optimization
* Database Security
* Backup and Recovery
* Distributed Databases
* NoSQL Databases
* Database Performance Tuning

## Technologies Used

* MySQL
* MariaDB
* MongoDB
* SQL
* XAMPP
* phpMyAdmin
* MongoDB Compass

## Practical Exercises

* Creating and Managing Databases
* Writing Complex SQL Queries
* Implementing Stored Procedures and Triggers
* Working with Transactions
* Query Optimization Techniques
* Database Security Implementation
* MongoDB CRUD Operations
* Performance Analysis

## Software Requirements

* MySQL Server 8.x or later
* MongoDB Community Server
* XAMPP
* MySQL Workbench or phpMyAdmin
* MongoDB Compass

## Learning Outcomes

After completing this module, students will be able to:

* Design efficient database systems.
* Develop advanced SQL queries and database programs.
* Optimize database performance.
* Implement database security and recovery mechanisms.
* Work with both relational and NoSQL databases.
